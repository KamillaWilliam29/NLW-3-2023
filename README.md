# NLW-3-2023
Repository for the NLW event at September 2023

#O 3º evento do NLW deve acontecer dos dias 11 a 14 de setembro de 2023. Os projetos trabalharão com IA(Inteligência Artificial) voltada para programadores iniciantes. Da trilha a qual estou participando, a Foundantios, será iniciado o projeto do zero código a código, da instalação do VSCode aos comandos npm no terminal. Foram feitos mapas mentais para guiar e ajudar a galera que entrou no evento e não conhece nada sobre programação. 

Vamos até o final e #BoraCodar!
